Приложение написано для android API 31 на Java и OpenGL ES 2.0.
Класс MainActivity реализует обьект класса OpenGLView, который используется для настройки версии Opengl и создания обьекта
класса рендера MyGLRenderer и его связи с интерфейсом.
MyGLRenderer реализует логику отрисовывания сцены, созданием фреймбуферов, а также содержит в себе логику шейдеров.
Shape отрисовывает облака на текстуре, которая блендерится с текстурый земли в классе Sphere.
